# 길동이의 포트폴리오 (Gildong's Portfolio)

이 프로젝트는 React와 Vite를 사용하여 구축된 웹 포트폴리오 사이트입니다. 어느 Windows PC 환경에서도 쉽게 실행할 수 있도록 구성되었습니다.

## 📌 주요 특징 (프로젝트 설명)
- **프레임워크**: React (버전 18 컴포넌트 기반 웹 페이지 구축)
- **빌드 도구**: Vite (빠른 빌드와 개발 서버 제공)
- **특징**: Vite의 `usePolling` 옵션이 적용되어 있어 한글 폴더 경로나 특정 Windows 환경에서도 파일 읽기(lstat) 오류 없이 안정적으로 자동 빌드 및 실행이 가능합니다.

---

## 🛠 실행 전 필수 요구사항 (Requirements)
이 프로젝트를 실행하기 위해서는 기본적으로 사용자 PC에 **Node.js**가 설치되어 있어야 합니다.

1. **Node.js 설치**: [Node.js 공식 홈페이지(https://nodejs.org)](https://nodejs.org/)에 접속하여 최신 LTS 버전을 다운로드하고 설치합니다.
2. **설치 확인**: 설치가 완료된 후, `cmd`(명령 프롬프트) 또는 `PowerShell`을 열고 아래 명령어를 쳤을 때 버전 번호가 나오면 정상입니다.
   ```bash
   node -v
   npm -v
   ```

---

## 🚀 Windows 터미널에서 실행하는 방법

1. **터미널 열기**
   프로젝트 폴더(`package.json` 파일이 있는 위치)에서 상단 주소창을 클릭하고 `cmd`를 입력 후 엔터를 누르거나, 해당 폴더에서 마우스 우클릭 -> `터미널에서 열기(Open in Terminal)`를 선택합니다.

2. **패키지(Dependencies) 설치**
   아래 명령어를 입력하여 이 프로젝트 구동에 필요한 라이브러리들을 설치합니다. (최초 1회만 실행하면 됩니다.)
   ```bash
   npm install
   ```

3. **로컬 개발 서버 실행**
   설치가 끝났다면 아래 명령어를 입력하여 서버를 구동합니다.
   ```bash
   npm run dev
   ```
   이 명령어를 입력하면 자동으로 사용자의 기본 웹 브라우저가 열리며 포트폴리오 페이지(`http://localhost:5173/`)가 나타납니다. 

---

## 📂 주요 코드 구조 안내
- `package.json`: 이 프로젝트의 라이브러리 의존성(Requirements) 목록과 스크립트 명령어가 기록된 핵심 파일입니다. 파이썬의 `requirements.txt`와 동일한 역할을 수행합니다.
- `vite.config.js`: 프로젝트 빌드 및 실행과 관련된 설정 파일입니다. `server.open = true`를 설정하여 터미널에서 서버를 켜면 웹 브라우저가 자동으로 열리게 설정되어 있습니다.
- `src/`: 실제 포트폴리오의 화면을 구성하는 React 컴포넌트들(`.jsx`)과 스타일링 코드(`.css`)가 들어 있는 작업 폴더입니다.
